Του Ντούλα η μάνα

 Posted on October 16, 2017

Τίνος μανούλα θλίβεται ,τίνος μανούλα κλέγει,
του Ντουλα η μάνα θλίβεται ,του Ντουλα η μάνα κλέγει.
Σήμερα Ντούλα μ' Πασχαλιά , σήμερα πανηγύρι (μια Πασχαλιά και μια λαμπρή μια πίσιμην ημέρα),
αλλάζουν μάνες τα παιδιά κι οι πεθερές τις νύφες.
Κι 'συ Ντουλα μ' στη φυλακη, στα σκοτεινα μπουντρούμια (στις φυλακες στις σιδερένιες πόρτες).
Ντουλα μ' δεν αντριωνεσαι τα σιδερα να σπάσεις,
να πάρεις δίπλα τα βουνά,κι ολορθα τα λαγκάδια (δίπλα τα κορφοβούνια),
και να σφυρίξεις κλέφτικα να μάσεις παλληκαρια (να μαζωχτούν οι κλέφτες).
Να πάρεις σκλάβο πρόεδρο και τον εισαγγελέα, (να πάτε να πατήσετε του δικαστού το κάστρο)
που δε δικάζουν 'ξαμηνο,που δε δικάζουν χρόνο. (που δε δικάζει ισοβια, που δε δικάζει χρόνια)
Μόνο δικάζουν θανατον, κι ισοβια για 'σένα (μόνο δικαζει εις θανατον τα ομορφα παλληκάρια).
