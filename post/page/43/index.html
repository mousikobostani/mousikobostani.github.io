Παρουσίαση του Μουσικού Μποστανίου

 Posted on November 5, 2017

  1. Δημιουργία βίντεο που να παρουσιαζεί τα χαρακτηριστικά και τις δυνατότητες του Μουσικού Μποστανίου
    1. Εύκολη και γρήγορη εύρεση σημειώσεων βάση
      1. ονόματος
      2. ρυθμού
      3. δρόμου
      4. περιοχής
      5. συνδιασμός των παραπάνω (π.χ. καλαματιανό απο τη μακεδονία)
    2. Εύκολη δημιουργία σημειώσεων
      1. Το Emacs σε συνδιασμό με το org-mode (και άλλων πακέτων του emacs) μέσω των οποίων δημιουργήθηκε και αναπτύσεται το Μουσικό Μποστάνι αποτελούν υπεδύναμα εργαλεία. Είναι εργαλεία γραμμένα σε γλώσσα συναρτησιακού προγραμματισμού lisp1
    3. Εύκολη δημιουργία ιστοσελίδων 1.

Footnotes

1 Λένε οτι στο μέλλον όπου οι μηχανές θα βρίσκονται παντού και οι γλώσσες προγραμματισμού θα δίνουν και θα πέρνουν, το ίδιο θα ισχύει και με τους ιούς. Ο ανθρώπινος πολιτισμός θα φτάσει στο χείλος να καταστραφεί, και τότε θα έρθουν οι εξωγίηνοι να μας σώσους οι οποίοι μαντέψτε; Θα μιλάνε lisp. :smile:

Άμαν Ντόκτορ

 Posted on November 3, 2017

Αχ, δε μου λέτε πού \`ν’ αυτός ο ντόκτορ
που γιατρεύει τις πληγές, αμάν ντόκτορ,
που γιατρεύει τις πληγές.
Για να γιάνει και τις εδικές μου
που σαν τις μετρήσεις κλαις,
αμάν ντόκτορ.

Αμάν, οι πληγές μου μένα είναι μεγάλες
και δεν έχουν γιατρειά, αμάν ντόκτορ,
και δεν έχουν γιατρειά.
Η αγάπη μου με απαρνήθη
και έφυγε στα μακρυά,
αμάν ντόκτορ.

Αμάν, αμάν ντόκτορ, πες μου τι κάνω,
δεν αισθάνομαι καλά, αμάν ντόκτορ,
δεν αισθάνομαι καλά.
Πάρε γρόσια, πάρε όσα θέλεις
να μου γιάνεις την καρδιά,
αμάν ντόκτορ.

Τα γιατρικά που μού ‘δωσες γιατρέ μου
δέ με ωφελέσανε, αμάν ντόκτορ
δέ με ωφελέσανε
τις πληγές πού ‘χω στην καρδιά μου
τις χειροτερέψανε
αμάν ντόκτορ

Θέλω Να Γίνω Μπουφετζής

 Posted on November 3, 2017

Θέλω να γίνω μπουφετζής
σε τούρκικους τεκέδες
Να ‘ρχονται οι χανούμισσες
να πίνουν αργιλέδες

Πήγα κι εγώ κάποια βραδιά
και μπήκα στα μεράκια
Γνώρισα τρεις χανούμισες
τρία σωστά κουκλάκια

Η μια κρατά τον αργιλέ,
κι η άλλη το τσιμπούκι
Κι η τρίτη η μικρότερη
τρελή στο μαστουρλούκι

Ανοίξτε τα παράθυρα
να φύγουν τα ντουμάνια
Για να φουμάρουμε κι εμείς
που είμαστε χαρμάνια
